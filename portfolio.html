Section Funds

Goal

A 4th semester individual project to learn a new programming language and practice our software development methodologies.

Description

This tool takes formatted Excel reports and loads them as dataframes into a GUI. The reports are compared against a running spreadsheet of section spending. New, removed and updated records are displayed. The current iteration allows you to update the spreadsheet directly from the GUI for the first set of reports. The records for all reports are also output to formatted Excel files for easier reconciling.

Technology

  • Python
  • TKinter
  • Pandas
